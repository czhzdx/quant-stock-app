# 量化选股软件 - 使用说明

## 快速开始

### 方法一：双击启动（推荐）

1. 首次使用：双击 `安装依赖.bat` 安装所需依赖
2. 日常使用：双击 `启动应用.bat` 启动应用
3. 浏览器会自动打开 http://localhost:8501

### 方法二：命令行启动

```bash
# 安装依赖
pip install -r requirements.txt

# 启动应用
streamlit run web_app.py
```

## 系统要求

- Python 3.8+
- Windows / macOS / Linux

## 功能说明

- **数据源**: Akshare（支持A股/港股/美股）
- **策略**: 动量策略、双动量策略、均值回归策略、多因子策略
- **因子**: 100+技术指标和基本面因子

## 股票代码格式

| 市场 | 示例代码 |
|------|----------|
| A股 | 600519, 000858, 000333 |
| 港股 | 00700, 09988 |
| 美股 | AAPL, MSFT, GOOGL |

## 常见问题

**Q: TA-Lib安装失败？**

A: TA-Lib需要预编译包，Windows用户可从以下地址下载whl文件：
https://www.lfd.uci.edu/~gohlke/pythonlibs/#ta-lib

下载后运行：
```bash
pip install 下载的文件名.whl
```

**Q: 应用启动后浏览器没有自动打开？**

A: 手动访问 http://localhost:8501

**Q: 如何停止应用？**

A: 在命令行窗口按 `Ctrl+C`
